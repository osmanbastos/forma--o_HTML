Estrutura Básica do HTML

HTML é uma forma de organizar as informações através de marcações para criação de websites, cronstruindo sua estrutura através de blocos.

Falando sobre as Tags

Sendo assim, Ele funciona dividindo o texto através de "tags". TAGs são as marcas que servem também para dividir as informações em blocos, sendo essas as tags básicas:

  1. <html>
    2. Inicia uma página HTML. É necessário adicionar uma barra( / ) com a TAG </html> para sinalizar o fim da página. Todas as outras TAGs ficam dentro desta.
  2. <head>
    3. Adiciona um cabeçalho com informações a serem interpretadas pelo navegador O conteúdo desta TAG não aparece para o usuário.
  3. <body>
    4. É aqui onde é colocado todo o conteúdo que aparecerá para o usuário/cliente. Neste TAG serão inseridas outras TAGs para organização da página

Atributos das Tags

Atributos são características das TAGs que podemos alterar.

Exemplo:
<input type="text" id="seuTexto">
Neste exemplo a TAG de introdução de dados "input" recebe um atributo "type" que altera sua característica para "entrada de texto" e um outro atributo chamado "id" que adiciona uma identificação (seuTexto) à TAG, para que esta possa ser especificada em outro momento.

Textos

Os textos podem receber tratamentos especiais, como mudança da fonte <font> e alguns voltados a acessibilidade da página, como a TAG <strong>que dá enfase a tudo que estiver dentro dela</strong>, inclusive para leitura da página para pessoas com algum tipo de deficiência visual. Podemos colocar subilinhado com <u>, riscado com <del>. Também podemos encolher itens como o número '2' em H2O ou elevar como em x2 com as TAGs <sub> e <sup>.
